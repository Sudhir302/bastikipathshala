## Getting Started
### Prerequisites
- node
- VS code

### Installation 
1. Clone the bastikipathshala Repo:
    ```
    https://github.com/Sudhir302/bastikipathshala.git
    ```
2. Navigate to the project directory:
    ```
    cd client
    ```
3. Install dependencies:
    ```
    npm insall
    ```
4. Run the app 
    ```
    npm run dev
    ```
